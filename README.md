# GAS_CSV_extraction
当ツールはCSVデータを抽出するツールです。  
下記プロセスでの動作により、情報抽出を行うツールとなります。

1. 抽出先となるシート"テンプレート"のヘッダー部分を参照
2. 抽出元となるシート"CSV"のヘッダー情報と照合
3. 抽出先となるシート"テンプレート"に情報を転記
